# Путешествие по России

## О проекте

Данный проект представляет собой одностраничный сайт о местах для путешествий по России.

## Технологии

Применены следующие технологии: Grid Layout, Flexbox, БЭМ, Nested БЭМ, отзывчивая верстка.
